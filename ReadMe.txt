☆インストールの手順
　・ｓｅｔｕｐ．ｅｘｅを起動し、手順に従う。（インストールされたフォルダーを把握しておく必要有り）

※行列計算機はパソコン室での利用を目的としたため、パソコン室以外での動作確認は行うことができず、動作の保障をすることはできない

○行列計算機の利用方法
　  ・パソコン１台で利用する場合
　　　　　１：ＭａｔｒｉｘＣｕｌｃｕｌａｔｏｒを起動する。
　　　　　２：行列が何×何行列か記入し、行列表示ボタンを押す
　　　　　３：行列の要素に値を与える。
　　　　　４：記入した行列データをＡ〜の行列ボタンを押す事で、行列ボタンにデータを保存しておく
　　　　　５：モードを計算モードにする。
　　　　　６：行列ボタンと計算式ボタンを用いて計算式を立てる
　　　　　７：計算ボタンを押し、計算を行う

    ・分散して計算させる場合
　　　　　０：ＭａｔｒｉｘＣｕｌｃｕｌａｔｏｒを起動しておく
　　　　　１：ＭａｔｒｉｘＣｌｉｅｎｔを起動する
　　　　　２：サーバのパソコンのＩＰアドレスを記入し接続する。
　　　　　（いくつかの台数が接続を行う）
          ３、計算モードを並列・分散モードにする
　　　　　４：あとの手順は逐次の場合と同じである。
